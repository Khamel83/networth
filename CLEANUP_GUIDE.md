## Cleanup Commands

**IMPORTANT:** Review files before deleting! Backup if unsure.

```bash
# Remove backup files
rm -f index_backup.html

# Remove unused templates
rm -f templates/preferences.html
rm -f templates/login_simple.html
rm -f templates/index.html

# Remove old docs (if docs/ directory exists)
rm -rf docs/

# Remove development/test files
rm -f test_main.py
rm -f email_templates.py
rm -f welcome_template.py
rm -f create_templates.sh

# Verify nothing critical was removed
python3 validate.py
```

## After Cleanup

1. **Test locally:**
   ```bash
   ./quick-start.sh
   ```

2. **Validate:**
   ```bash
   python3 validate.py
   ```

3. **Commit:**
   ```bash
   git add .
   git commit -m "Clean up unused files"
   git push
   ```
