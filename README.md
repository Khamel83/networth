**Version**: 1.2

---

## What This Is

ONE_SHOT is a complete project-building methodology that transforms a 20-question questionnaire into a fully deployed application. It's not a project itself - it's a system for building projects autonomously using AI.

**Single file**: `one_shot.md` contains the entire workflow
**Target infrastructure**: OCI Always Free + Tailscale + GitHub ($0/month)
**Core principle**: Validate Before Create, Dependency Awareness, WHY documentation

---

## How It Works

### Phase 1: Interactive (5-15 minutes)
1. Answer 20 detailed questions about your project
2. Validate connections (GitHub, Tailscale, dependencies)
3. Make all architectural decisions upfront
4. Generate complete Product Requirements Document

### Phase 2: Autonomous (hours to days)
1. Create GitHub repository
2. Implement all features based on PRD
3. Write tests and documentation
4. Deploy to Tailscale HTTPS
5. Commit and push automatically

### Result
Working project at `https://[project].deer-panga.ts.net` with complete source code on GitHub.

## Architecture & Principles

### Validate Before Create
Always check task validity before creation. Environment validation prevents wasted effort.

### Dependency Awareness
Understand and document service dependencies. Prevent circular dependencies.

### Document the WHY
Why this database choice? Why this architecture? HOW changes, WHY remains constant.

### Progressive Enhancement
Build v1 that works. Iterate to v2 when needed. Don't over-engineer upfront.

### Free Infrastructure Default
Design for $0/month by default. Only add costs when absolutely necessary.
